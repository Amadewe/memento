  <body>
    {L'élément HTML <form> représente une section d'un document qui contient des contrôles interactifs permettant à un utilisateur d'envoyer des données à un serveur web.}
    <form>
    {L'élément HTML <fieldset> est utilisé afin de regrouper plusieurs contrôles interactifs ainsi que des étiquettes (<label>) dans un formulaire web.}
      <fieldset class="identity">
    {L'élément HTML <legend> représente une légende pour le contenu de son élément parent <fieldset>.}
        <legend class="identity">Votre identité</legend>
        <ol>
    {Le pseudo-élément ::placeholder représente le texte de substitution pour un élément de formulaire.}
          <li><label for="lastName">Nom :</label><input type="text" name="lastName" id="lastName/>" placeholder="Dupont"></li>
          <li><label for="firstName">Prénom :</label><input type="text" name="firstName" id="firstName/>" placeholder="Jean"></li>
          <li><label for="mailBox">Adresse mail :</label><input type="email" name="mailBox" id="mailBox/>" placeholder="Adresse mail"></li>
        </ol>
          </fieldset>
        </form>
        <form>
          <fieldset class="deliveryAddress">
            <legend class="deliveryAddress">Adresse de livraison</legend>
            <ol>
              <li><label for="adress">Adresse :</label><textarea rows="5" cols="14"></textarea></li> <!-- for et id doivent être identique -->
              <li><label for="postalCode">Code postal :</label><input type="text" name="postalCode" id="postalCode" placeholder="Code postal"></li>
              <li><label for="town">Ville :</label><input type="text" name="town" id="town/>" placeholder="Ville"></li>
            </ol>

                <input type="radio" name="radiol" value="radio1"><label> VISA</label><br/>
