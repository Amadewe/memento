La ligne de commande

Partie 1

  • Afficher la manuel de la commande Ls: man Ls
  • Afficher le contenu du dossier courant: Ls
  • Afficher le contenu du dossier courant avec les fichiers cachés: Ls -A
  • Afficher le contenu du dossier courant version longue sans les fichiers cachés: Ls -L
  • Afficher le contenu du dossier courant version longue avec les fichiers cachées: Ls -Al
  • Afficher le chemin du dossier courant: pwd
  • Se déplacer dans le dossier "Documents": cd Documents
  • Afficher le contenu de ce dossier: Ls

Partie 2

  • Se mettre en super utilisateur: sudo -s
  • Mettre à jour la liste des paquets: apt update
  • Mettre à jour des paquets: apt upgrade
  • Trouver tous les paquets contenant le mot XML: apt search XML
  • Installer le paquet git: apt install git
  • Avec le navigateur, télécharger le .deb à l'adresse http:/atom.io/ . Avec la ligne de commande, se déplacer dans le dossier Téléchargement. Lancer l'installation du .deb nouvellement téléchargé: dpkg -i atom-amd64.deb

Partie 3

  • Créer un dossier website dans le dossier Documents : mkdir -p website
  • Créer un dossier et un sous dossier en une seule ligne de commande dans le dossier website: Memento/assets: mkdir -p Memento/assets
  • Se déplacer dans le dossier nouvellement crée et faire trois nouveaux documents en une seule ligne css, img, js: mkdir -p css img js
  • Dans un dossier Memento créer le fichier style.css : touch style.css
  • Dans le dossier Memento créer le fichier index.html et copier coller un le texte de l'exercice: touch index.html
  • ouvrir nano: nano index.html
  • Déplacer le fichier style.css dans le dossier css du dossier assets: mv style.css /Documents/website/Memento/assets/css
note:
  • pour copier/coller Shift et inver
  • pour quitter et sauvegarder sur Nano: ctrl X

Nano exercice

  • Créer un dossier exerciceNano dans le dossier utilisateur : mkdir -p exerciceNano
  • Créer un fichier dans le dossier crée dans l'exercice 1 quisera nommé: index.html: touch index.html
  • Ouvrir le fichier avec nano: nano index.html et écrire dedans ceci:
    •   <!DOCTYPE html>
  <html>
  <head>Nano exercice</head>
  <body>
  <h1>Hello world !!!</h1>
  </body>
  </html>
  • Fermer: Ctrl X et vérifier le résultat dans Firefox: firefox index.html
  • Ajouter dans le fichier index.html la balise title et la remplir.
    •   <!DOCTYPE html>
  <html>
  <head>
  <title>titre de la page</title>
  </head>
  <body>
  <h1>Hello world !!!</h1>
  </body>
  </html>
  • Ajouter 3 titres h2 et les remplir.
    •   <!DOCTYPE html>
  <html>
  <head>
  <title>titre de la page</title>
  </head>
  <body>
  <h1>Hello world !!!</h1>
  <h2>titre 1</h2>
  <h2>titre 2</h2>
  <h2>titre 3</h2>
  </body>
  </html>
  • En se servant des options de nano remplacer les balises h2 par des paragraphes. Ctrl alt gr 8 pour remplacer. 
    •   <!DOCTYPE html>
  <html>
  <head>
  <title>titre de la page</title>
  </head>
  <body>
  <h1>Hello world !!!</h1>
  <p>titre 1</p>
  <p>titre 2</p>
  <p>titre 3</p>
  </body>
  </html>
